# Мэдээллийн сангийн дизайн

AgilePlatform нэг PostgreSQL 16 кластер дээр **таван тусгаарлагдсан схем** ашиглана — үйлчилгээ бүрт нэг. Энэ загвар нь монолит мэдээллийн сангийн үйлдлийн хялбар байдлыг тусдаа мэдээллийн сангийн хандалтын тусгаарлалттай хослуулна.
